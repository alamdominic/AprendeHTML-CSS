Cheatsheet HTML

Principales tags

Meta datos

  • <title> Titulo documento, aparece en la ventana
  • <meta> Información (charset, viewport, etc.)
  • <link> Links externos
  • <style> CSS
  • <script> Javascript

Texto y encabezado

  • <h1 al h6> Headings
  • <p> Párrafo
  • <br> Salto de línea
  • <hr> Línea horizontal
  • <pre> Texto preformateado

Contenido Agrupado

  • <div> Contenedor
  • <span> Contenedor en línea
  • <header> Encabezado de página
  • <footer> Pie de página
  • <main> Contenido principal
  • <section> Sección de contenido
  • <article> Artículo independiente
  • <aside> Contenido relacionado
  • <nav> Navegación
  • <a> Ancla, enlace: href="link", target="_blank"

Formato de texto

  • <strong> Negrita énfasis
  • <b> Negrita
  • <em> Énfasis
  • <i> Cursiva
  • <u> Subrayado
  • <mark> Texto resaltado
  • <small> Texto pequeño
  • <sub> Subíndice
  • <sup> Superíndice
  • <code> Código
  • <blockquote>
    Cita en bloque
  • <cite> Cita
  • <q> Cita corta (quoted)
  • <abbr> Abreviatura
  • <time> Fecha u hora

Multimedia

  • <img> Imagen src="imagen" alt="descripción" width="auto" height="auto"
  • <figure> Figura
  • <figcaption> Pie de imagen
  • <video> Video src="video" autoplay loop
  • <source> Fuente del video
  • <track> Subtítulos

Listas

  • <ul> Lista no ordenada
  • <ol> Lista ordenada
  • <li> Elemento de lista
  • <dl> Lista de definición
  • <dt> Término
  • <dd> Definición

Tablas

  • <table> Tabla
  • <tr> Fila de tabla
  • <th> Encabezado de columna
  • <td> Celda de datos
  • <thead> Encabezado de tabla
  • <tbody> Cuerpo de tabla
  • <tfoot> Pie de tabla
  • <caption> Título de la tabla

Formularios

  • <form> Formulario
  • <input> Entrada de datos
  • <label> Etiqueta de campo
  • <textarea> Área de texto
  • <select> Menú desplegable
  • <option> Opción dentro de <select>
  • <button> Botón
  • <fieldset> Agrupar campos
  • <legend> Título del fieldset
